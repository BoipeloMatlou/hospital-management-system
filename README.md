# hospital-management-system
A C# console application that manages patient records across multiple hospital wards using object-oriented programming.

📋 Description
>This program simulates a basic hospital management system where staff can add patients to specific wards and view all current patients. Built as part of my C# OOP coursework at Eduvos.

✨ Features
>Add patients to a specific ward with name, age, and condition details

>Display all patients organised by ward

>Handles ward capacity limits (max 4 patients per ward, 3 wards total)

>Validates ward numbers and handles invalid input

>Menu-driven interface for ease of use

🛠️ Built With
>Language: C# | Platform: .NET Console Application

🚀 How to Run
>Clone the repository

>Open the project in Visual Studio or any C# IDE

>Build and run HospitalManagementSystem.cs

>Use the menu to add patients or display current ward occupancy

💡 Concepts Demonstrated
>Object-oriented programming (classes, objects)

>2D arrays of objects

>null checking for empty slots

>switch statements and do-while loops

>Constants and encapsulation

>Multiple class structure (Patient, Hospital, Program)
